# QSkill Internship Project

## Overview

This project was developed as part of the QSkill Front-End Development Internship.

The application contains:

* Language Translator using RapidAPI
* Random String Generator
* Client Side Routing using React Router DOM
* Modern UI using Tailwind CSS

---

## Features

### Translator Application

* Translate English text into Hindi
* API Integration using RapidAPI
* Responsive User Interface
* Error Handling

### Random String Generator

* Generate Random Strings
* Built using:

  * useState
  * useEffect
  * useCallback

### Routing

* Home Page
* Translator Page
* Random Generator Page

---

## Technologies Used

* React
* Vite
* Tailwind CSS
* Axios
* React Router DOM
* RapidAPI

---

## Installation

Clone Repository

git clone YOUR_REPOSITORY_LINK

Move into Project Folder

cd qskill-project

Install Dependencies

npm install

---

## Environment Variables

Create a file named:

.env

Add:

VITE_RAPID_API_KEY=YOUR_API_KEY

---

## Run Project

npm run dev

Open:

http://localhost:5173

---

## Build Project

npm run build
